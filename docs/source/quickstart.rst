
Быстрый старт
=============

Обработка данных
----------------

1. Поместите исходные данные в ``data/source/``

2. Запустите пайплайн обработки::

    make dvc-repro

Или поэтапно::

    # Загрузка данных
    uv run python -m sports_forecast.data.ingest

    # Очистка и создание признаков
    uv run python -m sports_forecast.features.clean

Работа с документацией
----------------------

Сборка документации::

    make docs

Запуск сервера с автообновлением::

    make docs-serve

Разработка
----------

Форматирование кода::

    make format

Проверка линтером::

    make lint

Автофикс проблем::

    make fix
